# Setup Instructions for Land of Ghost

## Prerequisites
Before you begin, ensure you have the following installed:
- **Git**: [Download Git](https://git-scm.com/downloads)
- **Node.js**: [Download Node.js](https://nodejs.org/)
- **npm**: npm is included with Node.js installation.

## Cloning the Repository
First, clone the repository to your local machine:
```bash
git clone https://github.com/nehls01/land-of-ghost.git
cd land-of-ghost
```

## Installing Dependencies
Once you have the repository on your local machine, navigate to the project directory and install the required dependencies:
```bash
npm install
```

## Running the Application
To start the application, use the following command:
```bash
npm start
```
```
## Additional Information
- Ensure you have the correct version of Node.js for compatibility with the project.
- For any issues during setup, please reach out to the maintainers for assistance.
