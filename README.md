# Land of Ghost Isle Evrima Server

## Overview
Welcome to the Land of Ghost Isle Evrima server! This server is dedicated to providing players with an immersive and engaging experience in the world of Evrima. Explore, survive, and thrive in a vibrant community.

## Features
- **Friendly Community**: Join our welcoming and active community of players.
- **Unique Gameplay Mechanics**: Experience special server rules and settings that enhance gameplay.
- **Events**: Participate in regular events, contests, and challenges for exciting rewards.

## Getting Started
1. **Join the Server**: Use the following IP address to connect to the server: `your.server.ip`
2. **Create Your Character**: Follow the in-game prompts to create a new character and start your adventure.
3. **Explore the World**: Discover various biomes, quests, and new friends as you navigate the Land of Ghost Isle.
